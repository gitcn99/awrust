# cc-core 示例

## 示例列表

### basic_config - 配置加载

```bash
cargo run --example basic_config
```

### mysql_connect - MySQL 连接

```bash
cargo run --example mysql_connect
```

### redis_connect - Redis 连接

```bash
cargo run --example redis_connect
```

## 配置

复制 `config/config.toml.example` 为 `config/config.toml` 并修改连接信息。
